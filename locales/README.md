# i18n Locales (Nuxt 3)

This `/locales` directory contains all translation files used by the app’s internationalization (i18n) system.

The goal is to keep **all user-facing text** centralized, consistent, and easy to maintain across multiple languages.

---

## Folder Structure

Each folder represents a locale code:

├── en/
├── ru-RU/
├── uz-Cyrl-UZ/
├── uz-Latn-UZ/
└── README.md

### Locale meanings

- `en` → English (generic)
- `ru-RU` → Russian (Russia)
- `uz-Cyrl-UZ` → Uzbek (Cyrillic script, Uzbekistan)
- `uz-Latn-UZ` → Uzbek (Latin script, Uzbekistan)
